Primary Aim

To determine whether a given set of vectors in ℝn is linearly dependent or linearly independent.
To apply Gaussian elimination and rank computation as the core method for the test.
To interpret the result both algebraically and geometrically with real examples.

Theoretical Foundation

Understanding Linear Dependence and Independence

Core Idea

A set of vectors {v₁, v₂, …, vₖ} is linearly independent if no vector can be written as a combination of the others — each one points in a genuinely new direction. If at least one can be written as a combination of the others, the set is linearly dependent (it carries redundant information).

Formal Definition

The set is linearly dependent iff there exist scalars c₁,…,cₖ not all zero with

c₁v₁ + c₂v₂ + … + cₖvₖ = 0

It is linearly independent iff the only solution is the trivial one c₁ = c₂ = … = cₖ = 0.

Key Results

Rank Test (general)

Form matrix A with the vectors as columns. Then

rank(A) = n ⇒ Independent    |    rank(A) < n ⇒ Dependent

where n is the number of vectors.

Determinant Test (square only)

For n vectors in ℝⁿ: det(A) ≠ 0 ⇒ independent; det(A) = 0 ⇒ dependent.

Useful Facts

• Any set containing the zero vector is dependent.
• In ℝⁿ, more than n vectors are always dependent.
• A basis of ℝⁿ is any independent set of exactly n vectors.

Method — Gaussian Elimination

Build matrix A with the vectors as columns (or rows).
Reduce A to row-echelon form using elementary row operations (swap, scale, add multiple of one row to another).
Count non-zero rows = rank.
Compare with n: rank = n → Independent; rank < n → Dependent.
Quick Example

v₁=(1,2), v₂=(2,4) in ℝ². det = 1·4 − 2·2 = 0 ⇒ Dependent (v₂ = 2·v₁).

v₁=(1,0), v₂=(0,1). det = 1 ≠ 0 ⇒ Independent — the standard basis of ℝ².

Geometric Picture

• Two vectors are dependent iff they lie on the same line through the origin.
• Three vectors in ℝ³ are dependent iff they lie in the same plane through the origin.
• Independent vectors span a subspace whose dimension equals their count.

Why It Matters

Used in PCA & feature selection (ML), basis construction (graphics), independent loop equations (circuits), error-correcting codes, and any system of linear equations where we need to know if information is redundant.

Linear Dependence Simulator

A guided, step-by-step lab for testing dependence & independence of vectors

Input n vectors Form matrix A Row reduce Find rank Compare with n
1 Vector Dependence Test

Enter n vectors of dimension d. The simulator forms matrix A, performs Gaussian elimination row-by-row, finds the rank, and compares it with n.

Build the grid, fill in the vectors, then click Run Step-by-Step Simulation.

2 Characteristic Equation Check

Enter a characteristic equation in λ (or x). The simulator solves it, builds an eigenvector set (distinct λ → independent basis vector; repeated λ → forced repeat), forms matrix A, performs Gaussian elimination, and reports the verdict.

Enter a characteristic equation and click Run.
